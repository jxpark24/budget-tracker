# budget-tracker
A simple open-source budget tracker to manage income and expenses.

# Motivation
My mom keep telling me I tend to spend much money for my car BMW 330i, but I would like to prove that I AM ACUTALLY NOT.

# Requirments
In order to run this program you need to create your own data file.
(cp data.sample.json data.json) or manually add data.json. data.json stores your personal transctions and is ignored by Git.
